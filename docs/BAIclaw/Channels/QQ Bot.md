# QQ Bot


1. 创建机器人
	- 进入 [快速注册创建QQ机器人](https://q.qq.com/qqbot/openclaw/login.html)
	- 点击**创建机器人**
	- 在机器人页面中找到 **AppID** 和 **AppSecret**，分别点击右侧复制按钮，保存到记事本或备忘录中。**AppSecret** 不支持明文保存，离开页面后再查看会强制重置，请务必妥善保存。
![](Pasted%20image%2020260319003627.png)
2. 在 BAIclaw 中添加 QQ Bot
	- 输入 **AppID** 和 **AppSecret**
