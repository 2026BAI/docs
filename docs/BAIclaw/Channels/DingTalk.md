# DingTalk

1. **第一步：点击 “创建或加入企业”**

![](Pasted%20image%2020260319004613.png)

- 点击 “企业” 后，选择 “创建企业/团体”
![](Pasted%20image%2020260319005014.png)

- 填写信息并创建（填写 “企业名称” 后会出现如下内容需要填写，企业名称需要超过三个字，内容可随便填写，无特殊要求）
![](Pasted%20image%2020260319005321.png)

2. **第二步：进入[钉钉开发者平台](https://open-dev.dingtalk.com)创建机器人**

- 选择刚刚创建的组织
![](Pasted%20image%2020260319005820.png)

- 选择 “创建”
![](Pasted%20image%2020260319010139.png)

- 左侧栏目选择 “机器人” 后，点击 “创建应用”
![](Pasted%20image%2020260319010213.png)

- 填写 “应用名称” 和 “应用描述” 后点击 “保存”
![](Pasted%20image%2020260319010557.png)

- 在左侧栏目选择 “添加应用能力” 并选择添加“机器人”
![](Pasted%20image%2020260319010802.png)

- 开启机器人配置并填写内容
![](Pasted%20image%2020260319010941.png)
![](Pasted%20image%2020260319011156.png)


3. **第三步：调整权限后发布即可绑定**
- 在左侧栏目选择 “权限管理”，搜索 “消息”，勾选图中两项后点击 “批量申请”
![](Pasted%20image%2020260319011512.png)

- 在左侧栏目选择 “版本管理与发布” 后，点击创建新版本，必须创建版本信息才能够发布
![](Pasted%20image%2020260319011628.png)
- 输入”应用版本号“ 和 “版本描述” 后**保存**，即发布完成
- 选择左侧栏目的 “凭证与基础信息”，复制 APP ID 和 Client Secret 到 BAIclaw 客户端中保存并连接即可完成互联

4. **第四步：搜索创建的机器人**
- 在钉钉【消息】栏的“搜索框”搜索刚刚创建的 “机器人名称”，会展示在功能栏下
