
# Telegram

1. 前提：已有 Telegram 账号
2. 创建 Bot 并获取 Bot Token
	- 打开 Telegram，搜索并打开 **@BotFather**（带认证）
	- 发送：`/newbot`
	- 按提示输入并创建 bot（**必须以 `_bot` 结尾**，例如 `myclaw123_bot`）
	- 成功后复制 **Bot Token**（格式：`123456:ABCdef...`）并粘贴在 **机器人令牌** 输入框中

2. 获取用户ID
	- 搜索并打开 **@userinfobot** 并点击开始
	- 从 @userinfobot 获取您的用户 ID 并粘贴在 **允许的用户ID** 输入框中
	- 在 Telegram 搜索你刚创建的bot 并进行对话。

# Discord


1. 前提：有 Discord 账号、管理权限的服务器
2. 前往 [Discord 开发者门户](https://discord.com/developers/home) → 应用 → 新建应用
![](Pasted%20image%2020260318015402.png)

3. 在“Bot”部分：添加Bot，然后复制机器人令牌(Bot token)，添加到产品页面，并填写目标server id 和 channel id 

![](Pasted%20image%2020260318015134.png)

![](Pasted%20image%2020260318015147.png)


4. 在 Bot → Privileged Gateway Intents 中启用 Message Content Intent 和 Server Members Intent
![](Pasted%20image%2020260318015231.png)

5. 在 OAuth2 → URL Generator：选择 "bot" + "applications.commands"，添加消息权限
![](Pasted%20image%2020260318015243.png)

6. 使用生成的 URL 邀请机器人加入您的服务器
![](Pasted%20image%2020260318015258.png)


# WhatsApp

1. 在手机上打开 WhatsApp
    
2. 前往 设置 > 已关联设备 >包含连接设备
    
3. 扫描ClawX中显示的二维码
    
4. 系统将自动识别您的手机号
